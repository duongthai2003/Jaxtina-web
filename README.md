# Jax Learning English

React TypeScript project built with Vite.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```

### Preview

Preview production build:

```bash
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- ESLint
